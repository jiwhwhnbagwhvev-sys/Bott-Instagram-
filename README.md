# Bott-Instagram
Secure Terminal OTP App (BOTT) adalah aplikasi berbasis Terminal (Termux/Linux) yang dirancang untuk mensimulasikan dan mengimplementasikan sistem login aman serta verifikasi OTP real-time melalui Telegram dengan tampilan terminal yang profesional dan interaktif.
Aplikasi ini dibuat untuk pembelajaran, pengembangan skill backend, dan simulasi sistem keamanan, bukan untuk spam atau penyalahgunaan layanan.
